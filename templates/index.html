fediv

This is a service that provides badges for Fediverse (ActivityPub and AT Protocol).

API Usage

You can use this service by embedding the following URL in your HTML or Markdown:

<img src="https://img.shields.io/endpoint?url={{ fdv_host }}/api/<protocol>/<endpoint>?account=<your account>">

Protocols and Endpoints

ActivityPub

  • followers: Followers count
  • following: Following count
  • statuses: Statuses count

AT Protocol

  • followers: Followers count
  • following: Following count
  • posts: Posts count
  • profile: Profile

Example

To get the followers count for the account @example@example.com on an ActivityPub service:

<img src="https://img.shields.io/endpoint?url={{ fdv_host }}/api/activitypub/followers?account=@example@example.com">
